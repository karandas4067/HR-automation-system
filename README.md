# HR-automation-system
The HR Automation System is designed to simplify and automate various human resource management tasks such as employee data management, attendance tracking, leave management, and recruitment workflow. The project reduces manual work by digitizing repetitive HR operations, ensuring accuracy, consistency, and efficiency in managing employee records.

# Key Features:

Employee Management: Add, update, and manage employee profiles with complete personal and professional details.
Attendance Tracking: Automates daily attendance recording and report generation.
Leave Management: Enables employees to apply for leave and HR to approve/reject with real-time updates.
Recruitment Automation: Streamlines candidate shortlisting and onboarding processes.
Report Generation: Automatically generates reports on employee performance, attendance, and payroll.

# Technologies Used:

Frontend: HTML, CSS, JavaScript (or React if you used it)
Backend: Python (Flask/Django) or Java (Spring Boot) or Salesforce (if built there)
Database: MySQL / PostgreSQL / Firebase
Tools: GitHub, VS Code

# Outcome:
The HR Automation System enhances productivity, reduces paperwork, and minimizes human errors in HR processes, making the overall management system faster and more reliable.
